# Property Agent Management App

This project is a simple full-stack TypeScript application demonstrating:
- Relational data modeling
- REST API development with Node.js + Express
- In-memory data storage
- Vue.js frontend for upserting Property Agents

## Tech Stack
- Backend: Node.js, Express, TypeScript
- Frontend: Vue 3, TypeScript
- Storage: In-memory (no database)

## Structure
- `/backend` – REST API
- `/frontend` – Vue client
- `/docs` – ER Diagram

## Stretch Goal
- Input validation and email uniqueness enforcement
